# Portfolio

Sito portfolio personale statico per presentare i lavori di design in modo chiaro, coerente e minimale. Progetto pensato per valutazione accademica.

## Obiettivi
- Mostrare i lavori in modo chiaro e ordinato
- Comunicare identità e stile come designer
- Mantenere coerenza visiva e semplicità

## Struttura del progetto
Struttura da mantenere invariata:

portfolio/
- index.html
- bio.html
- portfolio.html
  - lavoro-1.html
  - lavoro-2.html
  - lavoro-3.html
- contatti.html
- css/
  - style.css
- js/
  - main.js
- img/
- README.md
- CHANGELOG.md

## Pagine
- `index.html`: home
- `bio.html`: biografia e competenze
- `portfolio.html`: lista/galleria di tutti i lavori
- `lavoro-1.html`, `lavoro-2.html`, `lavoro-3.html`: pagine dedicate ai singoli progetti
- `contatti.html`: contatti

## Tecnologie utilizzate
- HTML5
- CSS3
- JavaScript (vanilla)

## Design system
### Colori (solo questi)
Primari:
- #FFFFFF (bianco)
- #000000 (nero)
- #FF0000 (rosso)
- #0000FF (blu)

Secondari:
- #565656 (grigio 1)
- #9E9E9E (grigio 2)

### Tipografia
- Google Fonts: "Fragment Mono"
- Varianti consentite: Regular e Italic

## Linee guida
- HTML semantico, struttura pulita e accessibile (alt, label, ecc.)
- Un solo foglio di stile: `css/style.css` (no inline style)
- JavaScript minimo e mirato in `js/main.js` (niente framework/librerie)
- Mantieni il sito leggero e ottimizza le immagini quando possibile

## Avvio
Apri `index.html` nel browser.
